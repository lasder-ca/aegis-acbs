# Real-map benchmark fixture

`hatfield-uk.osm` is a compact road extract around Hatfield, Hertfordshire, United Kingdom. It is derived from OpenStreetMap data and is included solely as a deterministic smoke-test and demonstration dataset.

- Data attribution: © OpenStreetMap contributors
- Data licence: Open Data Commons Open Database License 1.0 (ODbL)
- Software licence: the Aegis source code is MIT licensed; this data file remains under ODbL.
- Approximate bounds: 51.7600–51.7700 N, 0.2420–0.2280 W

Do not use this small fixture to draw conclusions about nationwide routing performance. Use the city download manifests or DIMACS road networks for serious benchmark runs.
